相关概念：

1. 认证：
	一个用户能不能进入某一个系统，需要经过认证。
	认证方式有：登录，扫描瞳孔，刷脸，声音。

2. 授权：
	一个用户首先经过认证之后，能进入系统了，能进来并不意味着任何资源都能使用。
	到底能使用什么资源，不能使用什么资源，就必须经过“授权”才能得知
